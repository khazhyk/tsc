the htm file has the main info

teams.csv has the teams
matches.csv has the matches

Paste the data into the appropriate text boxes, then click "import data".
Click "calculate elo" to calculate the elo. Clicking it multiple time won't do anything.

Elo is stored internally as a decimal, but is truncated when displayed for cleanliness.


Click "show scores" to show the scores. By default it's sorted by score. Click on a column header to sort by that column.

Click "show matches" to show the matches. By default it's sorted chronologically.
